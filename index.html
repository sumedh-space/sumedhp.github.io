I make sure security is built into the product, not bolted on afterwards — from reviewing architecture and threat-modelling new features, to hardening cloud infrastructure, managing vulnerabilities, and embedding security checks into CI/CD pipelines.
What tools and platforms do you work with?+
I work extensively with AWS, Splunk for SIEM and log analysis, secrets-scanning and supply-chain security tooling in CI/CD pipelines, and a range of vulnerability assessment and OSINT platforms.
